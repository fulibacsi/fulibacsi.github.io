Prelude to Python 101¶

This site will feature various materials to learn Python and the basics of programming, data cleansing, web scraping, visualising and - hopefully - many more.

Most likely you’re already aware of Python and (probably) you have some cumbersome task you want to automate, or a problem you’d like to solve. The intent behind this project is the lend you a hand and show some easy fix for them.
