How to install IPython in Windows

TL;DR:

  1. Install python.
  2. Make sure that python is in the system’s path.
  3. Make sure that python in in pythonpath.
  4. Download setuptools from: https://bitbucket.org/pypa/setuptools/raw/bootstrap/ez_setup.py
  5. Install setuptools from an administrator command line by executing python ez_setup.py.
  6. Install ipython by executing easy_install ipython[all] command from the your_python_directory\Scripts folder.
  7. Test it.
  8. In case of failure, download and install the appropriate version (highest version number, x86/amd64, python 2.7, .msi extension) of pyzmq from https://github.com/zeromq/pyzmq/downloads.
  9. Install jinja2 with easy_install: easy_install jinja2
  10. Install tornado as well: easy_install tornado
  11. If it is still fails with unicode error during executing notebooks, add IPYTHONDIR as a new environmental variable with a non-unicode directory (probably Python27\Scripts)
  12. Enjoy it.

In details:

1. Install python

Download the latest python version from: http://www.python.org/download/ Install it. Add python to system’s path. (how to: http://www.itechtalk.com/thread3595.html) Add pythonpath to the system. (how to: the same as above. If pythonpath is not in the list, create it.)

5. Install setuptools

Hit windows/super button. Start typing cmd Right click on cmd.exe > Run as administrator Go to the directory where the ez_setup.py is located. (Use the cd command to change directory) Execute python ez_setup.py.

7. Test it.

Go to your_python_directory\Scripts folder. Start iptest.exe

8. Enjoy it.

Use ipython.exe. Or use ipython notebook for the notebook version.
